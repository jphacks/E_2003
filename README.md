### 背景(製品開発のきっかけ、課題等）
　近年、日本では激甚災害が頻発している。そのような中、私たちが住んでいる熊本県において「熊本豪雨」が発生した。この災害により、球磨川が氾濫し、死亡者が出るほどの被害となった。
 
 しかし、今回の被害はハザードマップで事前に予測されていた。このことから、ハザードマップは防災に有効なコンテンツであることが窺えるが、事前に確認されておらず、「分かりづらい」などの意見もあった。
 
 そのため、私たちは浸水の危険性を示すコンテンツとして「ARERT」というiPad向けのアプリを開発し、リリースした。※以下URL参照
https://apps.apple.com/us/app/arert/id1512900814
